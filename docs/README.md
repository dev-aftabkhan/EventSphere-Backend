# Documentation — EventSphere Backend

This folder contains developer and operator documentation for the EventSphere backend.

Files:

- `ENV.md` — Environment variables required to run the server
- `SETUP.md` — Local setup and quickstart (Node + DB + Kafka)
- `ARCHITECTURE.md` — High level architecture and flow (Kafka, Mongo, MySQL)
- `KAFKA.md` — Kafka topics and Docker Compose used in the repo
- `API.md` — API endpoints, routes, controllers and short examples
- `CONTRIBUTING.md` — Guidelines for contributing and testing
- `OPERATIONS.md` — Operational notes, consumers, and scaling guidance

Start with `ENV.md` to configure your `.env` file, then follow `SETUP.md` to bring up dependencies.
